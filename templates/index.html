Endpoints

/api/question-extraction

POST endpoint

Uses a BERT Large Uncased model fine tuned on SQUAD 91.0% accuracy

Requires

  • question: "The question you want to ask"
  • context: "The passage of text you want to search in"

Returns

  • 'question': The question you asked
  • 'context': The context you passed
  • 'result': The answer to the question

/api/sentiment-extraction

POST endpoint

Finds the sentiment in the text

Requires

  • text: "The passage you want to analyse"

Returns

  • 'text': The passage you asked about
  • 'result': POSITIVE | NEGATIVE
